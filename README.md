# Pollify

Produkční URL: [pollify.vercel.app](https://pollify.vercel.app/)

Github repozitář: [github.com/vlkradek/pollify](https://github.com/vlkradek/pollify)

### Použité technologie

-	IDE – Visual Studio Code
-	Verzování – Git
-	Verzovací cloud – Github
-	Hosting - Vercel
-	Databázový GUI - DBeaver

### Použité technologie
-	Next.js – Fullstack framework běžící na node.js
-	Typescript – Nástavba javascriptu pro typování
-	SQL – Databázový jazyk
-	Postgres – Databázový systém
-	Tailwind CSS – CSS framework
